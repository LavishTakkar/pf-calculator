# PF Structure Calculator

A single-page tool to compare statutory PF (₹1,800/month) vs 12%-of-basic PF on
take-home pay, income tax (New Regime, FY 2026-27), and long-term wealth — EPF
corpus plus FD and equity mutual-fund alternatives at retirement.

Fully client-side, zero dependencies. Open `index.html` or visit the live page.
